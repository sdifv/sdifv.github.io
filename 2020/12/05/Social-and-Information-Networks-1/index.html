Social and Information Networks Ⅰ

社会网络信息分析课程知识点总结(Ⅰ)

graph theory / network representation / network model

图论

  • 分析网络的方式
    • 节点分类：预测给定节点的类型
    • 链路预测：预测两个节点是否存在链接
    • 社区检测：识别密集连接的节点集群
    • 社交影响/传播：预测共同的路径
    • 网络/节点相似性分析
  • 网络的组成
    • N: nodes/vertices
    • E: links/edges
    • G(N,E): network/graph
  • network 与 graph 的区别
    • network: 具体的系统
    • graph: 数学表示
  • 有向图与无向图
  • 网络的连通性()
    • undirected:
      • connected component
      • bridge edge
      • articulation point
    • directed
      • strongly connected directed graph：双向可达的有向图
      • weakly connected directed graph：连通的无向图

网络表示

  • Directed Graph

    • $In(v) = {w|w\ can\ reach\ v} \ \ Out(v)={w|v\ can\ reach\ w}$

    • two types

      • 强连通图：Strongly connected graphsS
      • 有向无环图：DAG – Directed Acyclic Graph

    • 强连通分量Strongly connected component (SCC)

      • 定义：任意两点可以互相可达的最大节点集合

      • 性质：

        1. SCC将图G进行分区，因此每个节点都应该属于且仅属于一个SCC(反证法)
        2. 如果图G由若干个SCC组成，那么图G是一个DAG

      • 将DAG划分为SCCs的方法：

        选取一个节点A，计算In(A),**Out(A)**，包含节点A的SCC计算公式为：$SCC(A)=Out(A)\cap In(A)$

  • network representations

    • edge list ： [(1,2),(1,4),…]

    • adjacency list ：{1:[2,4],4:[1,2,3],…}

    • Adjacency Matrix：$A_{ij}=1\ or\ 0$

  • 二部图Bipartite Graph

    • 定义：节点被划分为独立的两部分，两部分节点之间存在连接关系

  • Network Properties

    • 节点的度(degree) / 平均度 / 出(入)度

    • 节点度(概率)分布：$P(k) = N_k / N$， $N_k=$ nodes with degree k

    • 节点聚类系数(0,1)：$C_i = \frac {e_i} {k_i(k_i-1)/2}$，$k_i$ is the degree of node i，$e_i$ is the number of edges between the neighbours of node i

    • 平均聚类系数：$C=\frac 1 N \sum_i^NC_i$

    • 距离

      • 路径序列：$P_n = {i_0, i_1, …, i_n}$

      • 路径长度：$H_{uv}^{(h)} = [A^h]_{uv}$

    • 直径：图中任意两个节点的最大距离

    • 平均距离：图中任意两点之间的距离之和/$A_n^2$

网络模型

  • 简单图

    • Random Graph Model

      n个节点的无向图， edge(u,v)存在的概率为p，不唯一

      • 形成具有E条边的概率：

      $$
      P(E) = C_{E_{max}}^E p^E (1-p)^{E_{max}-E} \
      E_{max} = n(n-1)/2
      $$

      $P(E)$服从二项分布

      • 节点的平均度：$E[X_v]=\sum_{u=1}^{n-1}E[X_{vu}] = (n-1)p$
      • 节点度分布：$P(k) = C_{n-1}^k p^k (1-p)^{n-1-k}$，服从二项分布
      • 节点聚类系数：$C=p=k/n$
      • 虽然$G_{np}$是不现实的，但它是非常有用的！

  • 社交网络

    • 三元闭包Triadic Closure

      • span / bridges / local bridge

        • 桥(Bridge)：如果一个图中，已知A和B相连，若去掉连接A和B的边导致A和B分属不同的连通分量，则该边称为桥。桥为A,B间的唯一路径。
        • 捷径(Local bridge)：若边A-B的端点A和B没有共同朋友，则称边A-B为捷径，删除A-B边将把A-B距离增加至2以上(不含2)。
        • 跨度(Span)：边两端点在没有该边情况下的实际距离。

      • 强三元闭包

        如果A-B，A-C均为强联系，则B和C之间应该存在联系，否则违反强三元闭包性质。若节点A满足强三元闭包性质，并有至少两个强联系边与之相连，则与其相连的任何捷径均为弱联系。

      • 邻里重叠度

        NOL(A,B) = 与A和B均相邻的节点数/与A或B相邻的节点数

    • 图分割Graph Partitioning

      • 介数(betweenness)：一条边承载的信息流总量，即所有节点对之间最短路径上的流量之和。
      • 分割原则：逐步删除最高介数边

    • 介数计算的一种算法

    • 符号网络

      Signed networks: A complete graph is balanced if every connected triple of nodes has: All 3 edges labeled + or Exactly 1 edge labeled +
